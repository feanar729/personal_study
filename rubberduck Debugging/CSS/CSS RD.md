## CSS 선택자 특이성(Selector Specificity)은 무엇이며 어떻게 작동합니까?

- 브라우저는 CSS 규칙의 특수성에 따라 요소에 표시할 스타일을 결정한다. CSS 적용시 이러한 특정 규칙에 따라 우선순위가 부여되어 적용되는 것을 의미한다.
- 이를 쓰기 위한 이유는 CSS의 요소가 선언시 충돌을 피하기 위해 생긴 것이다.

- 작동방식은 다음과 같은데
> - 중요도 : 어디에 선언 되었는가?
>> - head 요소 내의 style 요소
>> - head 요소 내의 style 요소 내의 @import문
>> - <link>로 연결된 CSS 파일
>> - <link>로 연결된 CSS 파일 내의 @import문 
>> - 브라우저 default Style Sheet
> - 명시도 : 대상이 명확하게 특정되었는가?
>> - !importany > inline Style Attribute > id Selector > class = attribute = 가상선택자 > tag Element > 전체 선택자 > 상위 요소에 의해 상속된 속성
> - 우선순위 : 선언된 순서 위 -> 아래 순
>> - 기본적으로 맨 아래에 나온 property: value 가 우선순위가 높다
>> - 우선순위가 끝나면 선언 개수가 많은 것이 우선순위가 높다

- html body div.#foo div.foo span 순이면 이 샐렉터의 점수는? : 115점 

- CSS의 우선순위 값을 설명해보기 (ex:html body div#foo div.foo span)
